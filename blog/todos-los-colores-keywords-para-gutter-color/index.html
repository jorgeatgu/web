Todos los colores en keywords para Gutter Color

El 22 de abril de 2014

Archivado en: CSS3 SVG

Gutter color es un package para Sublime Text 3 el cual nos muestra el color que estamos utilizando en nuestros archivos CSS.

A algunos nos gusta utilizar los keywords disponibles para mostrar los colores, y como Gutter Color no compila los keywords, lo he solucionado creando un CSS con el nombre del color y su valor hexadecimal, así Gutter Color nos muestra en el lateral del archivo el color de cada keyword.
