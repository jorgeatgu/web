Web Animations en Google Chrome 36

El 23 de mayo de 2014

Archivado en: Javascript Web Animations

Ayer se publicó un post en HTML5 Rocks donde hablaban sobre la inclusión del element.animate() en Chrome 36.

Este anuncio es un pequeño empujon a la especificación, así que a partir de ahora no necesitaremos el Polyfill de Polymer para crear animaciones con element.animate() de Web Animations, eso sí por ahora solamente esta disponible en Google Canary, así que si vais a practicar seguid haciendolo con el polyfill de Polymer.

Por mi parte ayer hice un pequeño ejemplo modificando el que ya había hecho anteriormente para ilustrar el primer post que escribí sobre Web Animations 1.0

Recordad que el ejemplo por ahora solo funciona en Canary!!
