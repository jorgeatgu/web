Package Control. Donad malditos!

El 5 de enero de 2015

Archivado en: Sublime Text Package Control

Este verano mientras compilaba todos mis snippets de SVG en un plugin para el Package Control de Sublime Text descubrí algo que desconocía por completo. Y es que el famoso Package Control de Sublime Text es una creación de una persona totalmente ajena a Sublime Text.

El creador se llama Will Bond y como todo el mundo sabe el Package Control es totalmente gratuito. Desde el pasado verano Will Bond trabaja por su cuenta y acepta donaciones para por lo menos cubrir el mantenimiento de toda la estructura del Package Control. Que menos que hacer una pequeña aportación a algo que nos ha ahorrado tantas horas de trabajo.

Y si estais tiesos o simplemente sois los socios fundadores de la cofradía de la virgen del puño cerrado podéis colaborar dandole una estrella en GitHub.
